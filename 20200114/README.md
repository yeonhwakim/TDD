### 이해
 주어진 배열에서 3개수를 더한 값이 소수인 수의 개수를 구하라
 * 입력값: 숫자들이 들어있는 배열 nums가 매개변수
 * 출력값: nums에 있는 숫자들 중 서로 다른 3개를 골라 더했을 때 소수가 되는 경우의 개수를 return

 ### 계획
  1. 입력 값에서 모든 경우의 3개수를 뽑는다.
  2. 뽑은 값들을 더한다.
  3. 더한 값을 소수인지 아닌지 판단한다.
  4. 소수인 개수를 구한다.
